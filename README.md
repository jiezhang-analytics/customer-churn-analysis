# Netflix Customer Churn Prediction

This project analyzes customer churn for a Netflix-like streaming platform and builds machine learning models to identify users at risk of cancellation. The analysis combines data cleaning, exploratory analysis, churn prediction, customer segmentation, simplified CLV analysis, and retention recommendations.

## Business Objective

Streaming platforms lose revenue when subscribers become inactive and cancel. This project uses customer behavior, subscription, and account attributes to answer three questions:

- Which customer behaviors are most associated with churn?
- How accurately can churn be predicted from available customer data?
- What retention actions should the business prioritize?

## Dataset

- Source: [Kaggle - Netflix Customer Churn Dataset](https://www.kaggle.com/datasets/abdulwadood11220/netflix-customer-churn-dataset?resource=download)
- Rows: 5,000 customers
- Target: `churned`
- Key fields: `watch_hours`, `last_login_days`, `avg_watch_time_per_day`, `subscription_type`, `monthly_fee`, `number_of_profiles`

The dataset appears to be synthetic or sample data, so the conclusions should be treated as a portfolio analysis rather than production business evidence.

## Methods

- Data quality checks and preprocessing
- Exploratory data analysis with Pandas, Matplotlib, and Seaborn
- Scikit-learn preprocessing pipelines with `ColumnTransformer`
- Logistic Regression baseline model
- Random Forest comparison model
- Cross-validation and ROC-AUC evaluation
- K-Means customer segmentation
- Simplified subscription CLV analysis

## Key Findings

- Engagement behavior is the strongest churn signal.
- Customers with lower watch time and longer inactivity are more likely to churn.
- Basic plan users show weaker value and higher churn risk than higher-tier users.
- Multi-profile accounts tend to be more stable and valuable.
- Demographic variables are less useful than behavioral features for predicting churn.

## Business Recommendations

- Re-engage inactive users after 14, 30, and 45 days of inactivity.
- Target low-watch users with personalized recommendations and content reminders.
- Test upgrade incentives for Basic plan subscribers.
- Protect high-value household accounts with loyalty-focused retention actions.
- Use churn model scores to prioritize outreach, then validate campaigns with A/B tests.
